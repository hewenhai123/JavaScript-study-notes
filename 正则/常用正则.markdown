### 常用正则

|            用途         |                    正则                                            |
|:---|:---:|
| 有效数字验证        |     reg=/^[+-]?(\d&#124;([1-9]\d+))(\.\d+)?$/                           |
| 邮箱的             |     reg= /^\w+((-\w+)&#124;(\.\w+))*@[A-Za-z0-9]+((\.&#124;-)[A-Za-z0-9]+)*\.[A-Za-z0-9]+$/     |
| 电话的             |     reg = /^1\d{10}$/                                                    |
| 年龄在18-65之间的   |     reg = /^((18&#124;19)&#124;([2-5]\d)&#124;(6[0-5]))$/                |
| 中文姓名           |     reg = /^[\u4e00-\u9fa5]{2,4}$/                                        |
| 身份证             |     reg = /^(\d{6})(\d{4})(\d{2})(\d{2})(?:\d{2})(\d)(?:\d&#124;X)$/;     |
